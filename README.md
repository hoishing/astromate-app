# AstroMate

> A handy astrology app

<https://astromate.cc/app>

## Features

- Astrological Charts
  - Birth Chart
  - Synastry Chart
  - Transit Chart
  - Solar Return Chart
- Statistics of chart data
- PDF generation of charts
- Google OAuth for authentication
  - Save charts on server
- AI chat
  - BYOK (Bring Your Own Key)
  - Use any [OpenRouter models]
